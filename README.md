# 🛒 Система керування замовленнями

Проєкт на мові **C++**, розроблений для автоматизації обліку товарів у замовленнях. Програма підтримує роботу з форматом **JSON**, що дозволяє зберігати дані між сеансами роботи.

---

## 🚀 Основні можливості

* ➕ **Додавання товарів:** швидке додавання позицій із зазначенням ціни та кількості.
* ❌ **Видалення:** можливість видалити товар із кошика за його назвою.
* 📊 **Розрахунок вартості:** автоматичне обчислення загальної суми замовлення (Total).
* 💾 **Збереження та завантаження:** повна підтримка серіалізації в JSON через бібліотеку `nlohmann/json`.
* 🔄 **Статуси замовлення:** відстеження стану виконання (Pending, Processing, Completed, Cancelled).

---

## 📂 Структура файлів

| Файл | Опис |
| :--- | :--- |
| `product.h` | Структура товару та логіка конвертації в JSON. |
| `order.h` | Оголошення класу замовлення та переліку статусів. |
| `order.cpp` | Реалізація логіки управління замовленнями та робота з файлами. |
| `main.cpp` | Інтерактивне консольне меню для користувача. |

---

## 📝 Приклад використання

Після запуску ви побачите меню:
1. Оберіть пункт **1**, щоб додати продукт.
2. Введіть дані, наприклад: `Monitor 250 1`.
3. Оберіть пункт **3**, щоб побачити сформований чек.
4. Оберіть пункт **4**, щоб зберегти дані у файл `my_order.json`.

---

## 📄 Формат даних JSON
Файли замовлень зберігаються у зручному для читання форматі:
```json
{
    "orderID": 1,
    "orderDate": "2026-04-16",
    "status": 0,
    "items": [
        {
            "name": "Monitor",
            "price": 250.0,
            "quantity": 1
        }
    ]
}
